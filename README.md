# ProfilePage

## How to use

Download the example [or clone the repo](https://github.com/Wiolem/ProfilePage.git):

```sh
git clone https://github.com/Wiolem/ProfilePage.git
cd ProfilePage
```

Install it and run:

```sh
npm install
npm run start
```

## How to use gh-pages

```sh
npm run deploy
```
